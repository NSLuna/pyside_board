# PySide6 게시판 프로그램

PySide6 기반으로 구현한 데스크탑 CRUD 게시판입니다.
SQLite를 데이터베이스로 사용하며 QStackedWidget을 통해 페이지 전환을 구현했습니다.

이 프로젝트는 데이터디자인엔지니어링 과제전형 요구사항을 기준으로 설계&제작 되었습니다.

## 1. 기술 스텍
- Python 3.12.9
- PySide6
- DB : SQLite3
- 구조 : Qt Widgets / QStackedWidget 기반 화면 전환 구조

## 2. 기능개요

### CRUD기능
- Create : 제목, 내용, 작성자를 입력하여 새로운 게시글을 생성
- Read : 메인페이지에서 게시글 목록 조회와 더블클릭으로 상세 페이지로 이동
- Update : 상세페이지에서 수정 버튼 -> 수정페이지 진입
- Delete : 상세페이지 혹은 메인피이제이서 게시글 삭제 가능(QMessageBox로 삭제 확인 가능)

## 3. 프로그램 구조

### 전체페이지는 QStackeWidget 기반으로 구성
- MainPage - 게시글 목록 + 글쓰기 + 목록에서 삭제
- CreatePage - 게시글 작성 페이지
- DetailPage - 제목, 작성자, 내용조회, 수정/삭제/목록으로 돌아가기
- EditPage - 게시글 수정페이지

모든 페이지는 QWidget을 상속하여 개별 클래스로 분리되어 있습니다.

데이터 베이스 처리는 DBManager 클래스에서 일괄 관리합니다

## 4. 향후 개선방향 및 개발 의도

본 프로젝트는 PySide6를 활용한 첫 GUI 프로그램으로,
구현과정에서 다양한 UI/UX 구성 방식과 데이터 흐름 관리에 대해 학습하는데 중점을 두고 작업하였습니다

현재 버전은 PDF파일에 요구된 CRUD기능을 중심으로 구성되어 있으며,
아래와 같은 추가 기능 및 개선 사항을 통해 더 나은 방향으로 확장할 여지가 있습니다.

- 검색기능
- 페이지네이션 적용
- 삭제 후 일정 시간 Undo 기능
- QSS 기반 UI 스타일링 개선

프로젝트를 완성하며 구조 설계와 화면 전환 로직에 대해 많은 공부를 하였고,
